## Tech Stack (Planned)

- **Language:** Swift
- **UI Framework:** SwiftUI
- **Minimum macOS:** 14.0 (Sonoma)
- **Architecture:** MVVM
- **Distribution:** Mac App Store

## Getting Started

### Prerequisites

- macOS 14.0 or later
- Xcode 15.0 or later
- Apple Developer account (for distribution)

### Development Setup

```bash
# Clone the repository
git clone <repository-url>
cd music-companion-app

# Open in Xcode
open MusicCompanion.xcodeproj
```
